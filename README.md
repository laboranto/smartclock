# 스마트클락(smartclock)
간단한 JS 기반 시계 웹앱
- Openweathermap API를 이용한 위치 기반 날씨 기능 탑재
- 시스템 설정에 따른 다크모드 지원

### 사용 방법
1. OpenWeatherMap(https://openweathermap.org/)에 접속한 다음 'Get API Key' 버튼을 눌러 개인 키를 생성합니다. 이 키는 외부에 노출되어선 안 되며, 소중하게 보관해 주세요.
2. 'weather.js' 파일을 열면 "나의 API 키"라는 부분이 보일 겁니다. 여기서 '나의 API 키'라는 문장을 지우고 앞서 생성된 API 키를 큰따옴표 안에 붙여넣으십시오.
3. 'index.html'을 실행하시면 웹 브라우저를 통해 시계 화면이 나타납니다. 날씨 기능을 사용하시려면 위치 권한이 필요합니다.
