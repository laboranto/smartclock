# 스마트클락(smartclock)
간단한 JS 기반 시계 웹앱
- Open-Meteo를 이용한 위치 기반 날씨 기능 탑재
- 시스템 환경에 따른 다크모드 및 반응형 지원

### 사용 방법
- 'index.html'을 실행하시면 웹 브라우저를 통해 시계 화면이 나타납니다. 날씨 기능을 사용하시려면 위치 권한이 필요합니다.
